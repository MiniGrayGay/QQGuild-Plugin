## 说明
使用本插件可以让你的频道机器人使用云崽的功能！^_^

## 安装
```
 git clone --depth=1 https://github.com/2y8e9h22/QQGuild-Plugin ./plugins/QQGuild-Plugin
 pnpm install -P
```

## 配置
```
 # ./plugins/QQGuild-Plugin/config/config.yaml
 # 频道机器人的 APPID
 appId: ""
 # 频道机器人的 TOKEN
 token: ""
 # 沙盒测试
 sandbox: false
 # 接收全部消息，公域机器人必须设置为 false
 allMsg: false
```
